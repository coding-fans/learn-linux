==========
VirtualBox
==========

`VirtualBox <https://www.virtualbox.org/>`_ 是一款虚拟机软件，可以模拟出一台电脑(虚拟机)。

模拟出来的电脑称为 **虚拟机** ；运行 `VirtualBox <https://www.virtualbox.org/>`_ 软件的机器称为 **宿主** 。
用户可以像操作真实电脑一样操作虚拟机，包括 **安装操作系统** 。

因此，就算你只有 `Windows` 电脑，一样可以安装一个 `Linux` 系统用于学习。
虚拟机系统与宿主系统完全独立，不会相互影响，比 **双系统** 更有优势。
